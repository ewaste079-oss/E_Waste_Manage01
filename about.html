About Us

Welcome to our E-Waste Management System. In a world driven by technology, electronic waste is growing at an alarming rate. Our platform is built to address this critical issue by promoting responsible disposal, collection, and analysis of electronic waste.

This system allows individuals, organizations, and waste collectors to report, track, and manage e-waste in their localities. Through structured data collection and analytics, we help users make informed decisions and contribute to a cleaner environment.

Built using Python and Flask, the platform features a secure, user-friendly web interface. The backend supports real-time data storage using SQLite, while the admin panel allows monitoring of user submissions, trends, and contact messages.

  • Data-driven analytics dashboard
  • Interactive and responsive user interface
  • Secure database with structured storage (SQLite)
  • Admin access to manage data and users
